Intro


This website is based to be a guide for all Rust players to follow. It will explain all of the weapons and ammo types in detail. This will allow any current, or upcoming Rust player to be able to know what they need to get there hands on and in what stage of the game. Before reading, note that due to the new update there will be no recoil patterns or gun sounds. Here is a list of the sections in this website:

  • Intro - Introduction to the website

  • Weapons - All of the projectile weapons in the game

    • Military Grade

    • Homemade

  • Ammo - A list of the games ammo

  • Gallery - A gallery of all the images in the website

Ammo


This section of the article is used to display all of the ammo types in the game. There are no explenations on the ammo types as they are all very self-explanetory.



Rifle Ammo


5.56 Rifle Ammo


This ammo is the normal ammo which goes into all 5.56 weapons. It has no special effects.

5.56 HV Ammo


This ammo is high velocity ammo which goes into all 5.56 weapons. This means that the bulets travel faster.

5.56 Incendiary Ammo


This ammo is incindiary ammo which goes into all 5.56 weapons. This means that the bulets make fire on impact.

5.56 Exlosive Ammo


This ammo is explosive ammo which goes into all 5.56 weapons. This means that the bulets make an explosion on impact.
